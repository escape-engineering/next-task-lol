## [리그오브레전드 정보 페이지 바로가기](https://next-task-lol.vercel.app/)

> LIP는 **무료**로 **로그인 없이** 사용할 수 있는 리그오브레전드 정보 제공 서비스입니다.  
> **Next**와 **React**, **TypeScript**를 사용하여 개발하였으며
> 원하는 챔피언이나 아이템의 정보를 확인하고
> 이상형월드컵을 통해 인기 챔피언을 알아볼 수 있습니다!
